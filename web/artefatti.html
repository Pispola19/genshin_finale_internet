Pezzo sopra, assegnazione nel riquadro blu. AGGIUNGI / riga → Carica nel modulo → SALVA MODIFICHE.

Questo pezzo è equipaggiato: non puoi cambiare slot finché è su un personaggio. Per liberarlo, nel riquadro sotto scegli Solo magazzino (libero) e salva; poi potrai cambiare slot se serve.

Set e pezzo: suggerimenti dal catalogo; puoi scrivere set o pezzo nuovi: vengono normalizzati e salvati nel database per i prossimi utilizzi. Come si usa.

I suggerimenti sono in italiano (nome ufficiale). Puoi anche scrivere a mano (es. nome del client in inglese): al salvataggio viene ricondotto al pezzo ufficiale IT se riconosciuto. I cinque pezzi del set sono su slot diversi: per un set completo usa AGGIUNGI cinque volte cambiando Slot.
